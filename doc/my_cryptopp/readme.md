# readme

安装 readme 中有这个说明.

这个地方选取 tag 为 最新发表的tag

```shell

git clone https://github.com/weidai11/cryptopp.git
git checkout  -b tag_CRYPTOPP_8_9_0 CRYPTOPP_8_9_0

cd cryptop
make
make test
sudo make PREFIX=/usr/local/cryptopp install

```
